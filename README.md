# 📊 Sales Data Analysis Dashboard

## 🚀 Project Overview
This project focuses on analyzing sales data to extract meaningful business insights using Python.

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Folium (for map visualization)

## 📊 Key Analysis
- Monthly sales trend analysis
- Category-wise sales distribution
- Region-wise performance
- State-wise sales visualization using maps

## 📈 Key Insights
- Sales peak during year-end months
- Technology category generates highest revenue
- West region performs best
- Map visualization highlights high-sales states

## 📂 Files
- Jupyter Notebook (.ipynb)
- Charts & Visualizations

## 🌍 Output
Includes charts and map-based visualization for better decision making.
